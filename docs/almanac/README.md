# CodeAlmanac Python Wiki

This wiki documents the current Python rewrite of CodeAlmanac on `dev`. It is a
separate wiki root from `.almanac/`, built to test the opinionated structure in
`docs/plans/opinionated-wiki-structure.md`.

Read `pages/architecture/architecture-system-overview.md` first. It links to
the concept, decision, guide, and reference pages that explain the system.

## Writing Rules

- Every page starts with a lead paragraph.
- Every factual claim that depends on code or docs gets an inline citation.
- Every page links to related pages with `[[...]]`.
- Concept pages explain vocabulary.
- Architecture pages explain system shape.
- Decision pages follow the ADR-style standard.
- Guides follow the Diataxis how-to standard.
- Reference pages follow the Diataxis reference standard.
