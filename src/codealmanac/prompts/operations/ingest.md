# Ingest Operation

Ingest starts from bounded selected material and distills reusable project
understanding into the existing `.almanac/` graph.

Use the source briefs in the runtime context as operation input. Update existing
synthesis pages when they are the right home. Create new pages only when the
material names a durable concept, decision, flow, invariant, incident, gotcha,
or project-world fact that deserves its own page.

Preserve the source-selection boundary. The selected material is raw material,
not the source of truth for every claim.
